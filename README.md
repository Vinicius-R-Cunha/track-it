## Installing

### Clone this Repository
```bash
$ git clone https://github.com/Vinicius-R-Cunha/track-it.git
```

### Install dependencies

```bash
$ npm i
```

### Run the front-end with
```bash
npm start
```

### Or you can build the project running

```bash
$ npm run build
```
